# Lessons Learned

- SSH is the primary way Linux servers are managed remotely.
- RDP is commonly used for Windows administration.
- Static IPs can cause connectivity issues after network changes.
- ip a shows interface and IP information.
- ip route shows routing and default gateway information.
- Netplan manages network configuration on Ubuntu Server.
- Linux permissions are based on owner, group, and others.
- systemctl is used to manage services.
