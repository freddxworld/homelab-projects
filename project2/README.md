# Windows ↔ Linux Administration Lab

## Goal

Learn how to remotely manage Linux and Windows systems using SSH and RDP while practicing basic Linux administration.

## What I Did

- Created Windows and Ubuntu Server VMs in Proxmox
- Configured network connectivity between both VMs
- Enabled SSH access to Ubuntu Server
- Enabled RDP access to Windows
- Created Linux users and groups
- Managed file permissions and ownership
- Performed file transfers between Windows and Linux
- Practiced service management using systemctl

## Skills Demonstrated

- Networking
- SSH
- RDP
- Linux User Management
- Linux Groups
- File Permissions
- File Transfers
- Troubleshooting
